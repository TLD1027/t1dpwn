T1d的Python库，旨在为pwntools工具提供个人方便。

请注意，这个库主要是为T1d个人使用和方便而创建的。它包含一些基于pwntools的小方法，以满足个人需求。更新将基于T1d的可用性和兴趣进行。但是，如果有任何问题，T1d乐于接受来自其他开发者的反馈、建议和修正。欢迎和感谢各位开发者的想法和代码优化。期待与各位开发者的交流！
